# FRC Village Vikings 9289: AutoViz 🚀

**AutoViz** is a powerful and intuitive desktop application designed to simulate and visualize autonomous and teleoperated modes for FRC (FIRST Robotics Competition) robots, brought to you by the Village Vikings FRC Team 9289. By connecting directly to NetworkTables in real-time, AutoViz brings your robot code to life—without ever touching the field.

---

## Features

- **Real-Time Visualization**: See your robot’s behavior live as it updates via NetworkTables.
- **Autonomous & Teleop Modes**: Visualize both autonomous routines and driver control interactions.
- **Mechanism Support**: Simulate common FRC subsystems like shooting, intake, and drivetrain.
- **NetworkTables Integration**: Compatible with robot code using standard WPILib NT or NT4 formats.
- **Cross-Platform**: Desktop app built using Qt, available for Windows, macOS, and Linux.
- **No Simulation Setup Required**: Just run your robot code—AutoViz will visualize it automatically.

---

## How It Works

AutoViz connects to your robot program over the network using NetworkTables. It reads robot pose, heading, mechanism states, and control flags to render a real-time visualization of your robot on a 2D field.

You can run it alongside your robot code (real or simulated) to debug autonomous paths, tune teleop logic, or demo your robot's functionality to others—anytime, anywhere.
